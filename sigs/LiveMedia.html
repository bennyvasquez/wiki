# Live Media SIG

The Live Media Team is reponsible for AlmaLinux OS Live Media.

The Live Media SIG uses the SIG/LiveMedia (opens new window) chat channel for communication.

# Deliverables

  • ISO images download link: repo.almalinux.org/almalinux/8/live/x86_64 (opens new window).
  • Build scripts and configuration files: AlmaLinux/sig-livemedia (opens new window).
